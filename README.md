INSTRUCCIONES DE USO 

1.- Asegúrate de tener descomprimida la carpeta de archivos en la que se encuentra el programa. Yo te paso el archivo comprimido (.zip), y debe ser descompreso para poderse ejecutar el .bat de su interior. 
2.- Tener instalado Python. Simplemente puedes escribir en tu Terminal del PC "Python". Te llevará a la página de Microsoft store lista para la descarga. 
3.- Haz click en "Iniciar_juego", en la carpeta de archivos. Esto iniciará el juego en tu navegador personal.
